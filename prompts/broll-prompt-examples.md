# B-Roll Prompt Examples for Veo 3.1 Fast

These examples illustrate the prompt style for generating cinematic 9:16 B-roll via the Veo 3.1 API.
Each prompt targets the specific aesthetic of 1Commerce / Keith's PNW operator brand.

---

## operator_wisdom examples

### "Cathedral Principle" — long-term thinking
```
Cinematic vertical 9:16 shot. A lone craftsman's weathered hands laying stone blocks at dusk, Portland industrial warehouse district in background. Shallow depth-of-field, golden hour backlight, rack focus from hands to skyline. Slow push-in on a telephoto lens. Mood: deliberate, monumental, unhurried. No text. No people's faces. 8 seconds.
```

### "Systems operator" — control room aesthetic
```
Cinematic vertical 9:16. Extreme close-up of mechanical keyboard keys being typed, then slow pull-back to reveal a dual-monitor setup with code and dashboards glowing in a dark studio. Teal and amber accent lighting. Bokeh city lights visible through rain-streaked floor-to-ceiling windows. Mood: focused, late-night, solitary mastery. No text on screen. 8 seconds.
```

### "Revenue graph" — abstracted growth
```
Cinematic vertical 9:16. Time-lapse of a Pacific Northwest forest canopy from ground level looking up — trees swaying, light filtering through in shafts, clouds racing overhead. Telephoto compression. Desaturated with one warm highlight channel. Mood: organic growth, scale, patience. No faces. No text. 8 seconds.
```

## pnw_identity examples

### Cascade mountains — isolation aesthetic
```
Cinematic vertical 9:16 drone shot. Misty Cascade mountain range at dawn, low clouds hugging ridgelines, a single gravel logging road disappearing into the treeline below. Ultra-wide sensor, slow forward push. Color grade: cool teal shadows, warm amber highlights on peak. Mood: vast, solitary, Northwestern. No people. No text. 8 seconds.
```

### Portland industrial district — urban PNW
```
Cinematic vertical 9:16. Slick wet cobblestones in Portland's Pearl District after rain, neon signs reflected in puddles, a MAX train passing blurred in the background. Macro lens on the reflection, rack focus to mid-distance. Color grade: desaturated greens, electric blue reflections. Mood: gritty, rainy, alive. No people. No text. 8 seconds.
```


## Prompt construction checklist

When writing a new B-roll prompt, always include:

- [ ] Aspect ratio: **9:16 vertical**
- [ ] Duration: **8 seconds**
- [ ] Shot type (close-up, drone, wide, etc.)
- [ ] Subject (what's in frame — never faces)
- [ ] Lighting conditions (golden hour, overcast, neon, etc.)
- [ ] Camera movement (push-in, pull-back, static, dolly)
- [ ] Lens language (telephoto compression, bokeh, macro, etc.)
- [ ] Color grade vibe (teal-amber, desaturated, etc.)
- [ ] Mood / atmosphere (1-2 adjectives)
- [ ] Explicit: **No text in video**
- [ ] Explicit: **No faces** (unless pillar calls for environmental only)
